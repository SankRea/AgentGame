## 状态语义

- `vitality` / `heat`：身体和环境压力。
- `clarity` / `murmurs`：不是善恶值，而是叙述稳定性与声音侵入程度。
- `memory` / `fatherClues` / `truth`：玩家已拼合内容的维度。
- `guilt`：接触到的集体罪责，不等于胡安个人道德。
- `currentTimeLayer`：`embers`、`past`、`grave` 或 `subjective`。
- `currentPerspective`：当前叙述者，默认 `juan`。
