# 地点与时间层

| 地点 | 余烬层 | 往昔层 | 墓中层 |
| --- | --- | --- | --- |
| 下坡路 | 酷热、白光、远镇无声 | 多洛雷斯描述的绿色平原 | 阿文迪奥的话变成已经发生过的回声 |
| 科马拉空街 | 白墙、黑门洞、杂草 | 儿童、鸽群与黄昏声响的残影 | 低语从墙缝和地面聚集 |
| 爱杜薇海斯住所 | 黑暗房间与无人认领的家具 | 多洛雷斯的婚姻与离开 | 来客和主人都显出亡灵身份 |
| 教堂 | 钟声遥远，门常闭 | 伦特里亚的妥协与赦罪交易 | 祷词碎成无法抵达天堂的声音 |
| 半月庄记忆 | 地平线上的低矮阴影 | 土地、债务和暴力的权力中心 | 佩德罗的执念只剩沉重回声 |
| 墓地 | 干土、矮碑和塌陷入口 | 葬礼、马蹄与集体记忆 | 胡安与多罗特亚并置讲述 |
