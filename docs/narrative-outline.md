# 叙事纲要

## 核心视角

胡安·普雷西亚多遵照母亲多洛雷斯的遗愿来到科马拉，寻找从未履行父亲责任的佩德罗·巴拉莫。游戏不按年份复述历史，而让现实、亡灵低语、他人记忆与胡安死后的墓中谈话在同一地点重叠。玩家改变的是理解顺序、愿意聆听的对象与最终获得的叙事完整度，不改写人物既定命运。

## 纵向切片结构

1. 母亲的遗愿：照片、承诺与下坡路。
2. 下坡路：阿文迪奥指出科马拉和半月庄，也暗示自己同为佩德罗之子。
3. 空镇：爱杜薇海斯迎接胡安；空屋保存着离去者的家具与声音。
4. 半月庄的回声：账簿、土地与伦特里亚神父的罪责让佩德罗的权力逐渐成形。
5. 亡灵低语：达米亚娜的声音消失在街道上；胡安发现声音来自墙和土地。
6. 墓中：多罗特亚纠正胡安对自己死亡的叙述。玩家选择继续聆听，或带着已经拼合的记忆离开叙事。

## 非线性原则

- 同一调查点会在“余烬”“往昔”“墓中”呈现不同图像和文本。
- 说话者身份先以位置或声音标识，身份确认后才显示姓名。
- 佩德罗的权力通过土地、债务、赦罪和他人的恐惧间接呈现。
- 苏萨娜不是任务目标；她的海洋、身体和梦构成与科马拉完全不同的主观层。
- “胡安死于低语”既是反转，也是后半段叙事视角成立的条件。
