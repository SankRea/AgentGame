# 人物与关系

| 人物 | 游戏功能 | 关系与叙事边界 |
| --- | --- | --- |
| 胡安·普雷西亚多 | 玩家角色、听者、不可靠叙述者 | 多洛雷斯之子；寻找父亲；后来发现自己已在墓中讲述 |
| 佩德罗·巴拉莫 | 缺席的权力中心 | 胡安与阿文迪奥等人的父亲；半月庄主人；以债务、土地与暴力控制科马拉 |
| 多洛雷斯·普雷西亚多 | 遗愿与理想化故乡的来源 | 胡安之母；她记忆里的科马拉与眼前空镇互相冲突 |
| 阿文迪奥·马丁内斯 | 下坡路引路人 | 佩德罗之子；平静地说出科马拉与佩德罗的死亡 |
| 爱杜薇海斯·地亚达 | 第一位接待者与时间错位证据 | 多洛雷斯旧友；提前知道胡安到来；她自己的死亡起初不被点明 |
| 达米亚娜·西斯内罗斯 | 空街中的引导声音 | 曾照料胡安；她的突然消失让生死边界破裂 |
| 多罗特亚 | 墓中共同叙述者 | 纠正胡安关于死亡的说法；以寻找不存在的孩子度过一生 |
| 苏萨娜·圣胡安 | 不受佩德罗占有的主观记忆 | 佩德罗最深的执念；她的海洋记忆不服从科马拉或佩德罗的叙述 |
| 伦特里亚神父 | 宗教罪责与阶级妥协 | 明知暴行却受半月庄供养；对穷人与权贵采取不同的宽恕尺度 |
| 富尔戈尔·塞达诺 | 土地权力的执行者 | 管家；把佩德罗的欲望转译为账簿、产权与行动 |
| 米格尔·巴拉莫 | 继承暴力的亡灵 | 佩德罗之子；其死亡与不能安息的马蹄声成为重要回声 |
