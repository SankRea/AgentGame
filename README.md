# 佩德罗·巴拉莫：科马拉仍在说话 / Pedro Páramo: Echoes of Comala

一款以《佩德罗·巴拉莫》人物、关系和非线性叙事结构为基础的像素文学冒险纵向切片。玩家控制胡安·普雷西亚多进入科马拉，通过探索、聆听、调查和选择拼合关于母亲、父亲、土地与亡灵的记忆。

当前版本不是战斗 RPG。选择主要改变玩家能听见的声音、进入的时间层、对人物关系的理解和最终结局视角，不随意改写小说人物既定命运。

## 当前可玩内容

- 母亲遗愿与下坡路序章
- 一张 1680×1040 的程序化科马拉地图
- 科马拉入口、空街、爱杜薇海斯住所、教堂、墓地与半月庄远影
- 阿文迪奥、爱杜薇海斯、达米亚娜、伦特里亚神父和多罗特亚的声音
- 三个以上调查点、一次往昔层切换、胡安死亡的认知反转
- “又一个声音”“科马拉的记忆”“母亲所说的地方”和未说完分支
- 新主题独立存档、自动存档、手动存档与结局收藏
- 可收起 HUD、记忆簿、暂停页、对话打字机与对话记录
- 程序化干风、远钟、犬吠和随低语强度变化的音景

## 操作

- `WASD` / 方向键：行走
- `E`：交谈、调查、显示完整台词或推进对话
- 方向键 / `W` `S` / 数字键：选择对话回应
- `H`：对话中打开或关闭记录
- `Tab`：打开或关闭记忆簿
- `K`：手动保存
- `Esc`：暂停或返回

关闭音频不会阻断剧情。重要声音同时提供字幕、画面变化或交互提示。

## 技术栈

- Phaser 3.90
- TypeScript
- Vite
- Arcade Physics
- 960×540 逻辑分辨率
- JSON 驱动的对话、事件、遗物、遗愿、成就与结局
- LocalStorage 存档
- Phaser Graphics 程序化地图与像素占位角色
- Web Audio API 程序化环境声

## 内容结构

```text
src/data/comala/
├── manifest.json
├── story/initial.json
├── dialogue/
├── events/
├── endings/
├── items/
├── characters/
├── memories/
├── quests/
└── achievements/
```

`manifest.json` 注册章节、地图和内容文件。对话节点与选择使用稳定 ID；`scripts/validate-content.mjs` 检查节点跳转、NPC/事件/结局对话、物品奖励、任务链、选择 ID 唯一性和旧主题 ID。

## 存档格式

存档键为 `comala-whispers.save.v1`。保存内容包括章节、地图、玩家位置、已触发事件、已完成对话、故事状态、记忆与遗物进度、遗愿、成就、已解锁结局和音频设置。旧修仙 Demo 存档不会被读取。

## 开发命令

```powershell
npm ci
npm run validate:content
npm run build
npm run dev
```

仓库级代理指令禁止在本轮执行编译和测试，因此本次整改没有声称上述命令已通过；接手者应在允许执行后运行内容校验与构建。
