# 佩德罗·巴拉莫：科马拉仍在说话

**Pedro Páramo: Echoes of Comala**

一款使用 Phaser 3 与 TypeScript 制作的像素文学冒险游戏。玩家扮演胡安·普雷西亚多，循着母亲多洛雷斯临终留下的遗愿下到科马拉，在灼热的空街、废弃房屋、教堂和墓地之间聆听死者仍未说完的话。

游戏以探索、对话、调查、记忆拼合和叙事选择为核心，不包含传统战斗或等级成长。选择改变玩家能够听见的声音、时间层、叙事视角及结局，但不会随意改写原作人物的既定命运。

- 1680×1040 的连续科马拉地图
- 空街、废屋、爱杜薇海斯住所、教堂、墓地、枯井与半月庄远影
- 阿文迪奥、爱杜薇海斯、达米亚娜、伦特里亚神父和多罗特亚的声音
- 旧照片、无人家具、账册、墓地和回声等调查内容
- 现实、往昔覆影、主观记忆与墓中低语等时间层
- 胡安已经死亡的叙事认知反转
- “又一个声音”“科马拉的记忆”“母亲所说的地方”及“未说完的话”分支

## 视觉与声音

画面使用石灰白、尘土黄、干土褐、旧金与冷灰蓝构成现实、记忆和低语三套视觉层。地图和角色目前均由 Phaser Graphics 及程序化像素纹理生成。

- 标题画面从过曝正午逐渐衰变为黄昏
- 热浪、稀疏尘埃、记忆渗色与视野边缘低语
- 活人、亡灵和记忆人物采用不同或错位的影子规则
- 干风、远钟、犬吠、马蹄、低语与对话打字音
- 重要环境声提供左侧、右侧、前方、远处或地下的方向字幕

## 操作

| 按键 | 功能 |
| --- | --- |
| `WASD` / 方向键 | 行走 |
| `E` / `Enter` / 空格 | 交谈、调查、显示全文或推进对话 |
| `W` `S` / 上下方向键 | 选择对话回应 |
| 数字键 | 直接选择对应回应 |
| `H` | 打开或关闭对话记录 |
| `Tab` | 打开或关闭记忆簿 |
| `K` | 手动保存 |
| `Esc` | 暂停、返回或关闭面板 |

鼠标可用于标题菜单、对话选项、暂停页和设置面板。

## 设置与可访问性

标题界面和游戏暂停页均可打开设置面板。设置独立保存在本地，新建旅程不会清除这些偏好。

- 主音量、环境音量和对话音量
- 快速、标准或慢速文字
- 自动推进对话
- 镜头震动强度
- 热浪与轻微闪变开关
- 高对比文本
- 方向环境声字幕
- 全屏切换
- 返回标题与二次确认删除存档

## 存档

游戏使用浏览器 `LocalStorage`：

- 游戏存档：`comala-whispers.save.v1`
- 独立设置：`comala-whispers.settings.v1`

存档包含玩家位置、章节、故事变量、已触发事件、已完成对话、遗物、遗愿、成就、结局记录和当前设置。旧题材 Demo 的存档不会被读取。

## 技术结构

- Phaser 3.90
- TypeScript
- Vite
- Arcade Physics
- 960×540 逻辑分辨率
- JSON 驱动的对话、事件、物品、遗愿、成就和结局
- Phaser Graphics 程序化地图与角色纹理
- Web Audio API 程序化环境声

主要目录：

```text
src/
├── config/              # 游戏尺寸与视觉主题
├── data/comala/         # 科马拉章节内容包
├── effects/             # 热浪、尘埃、记忆渗色与低语效果
├── entities/            # 玩家和 NPC
├── maps/                # 科马拉地图
├── scenes/              # 启动、标题、游戏和结局场景
├── systems/             # 对话、事件、叙事状态、存档与音频
└── ui/                  # HUD、对话框、记忆簿和设置
```

`src/data/comala/` 中的内容使用稳定 ID 相互引用。内容校验脚本位于 `scripts/validate-content.mjs`。

## 本地运行
